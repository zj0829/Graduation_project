## 图2：数据流转时序图（竖版）

```mermaid
sequenceDiagram
    participant U as 用户浏览器
    participant B as FastAPI后端<br/>:8000
    participant D as DeepSeek/GLM<br/>外部API
    participant M as MCP Gateway<br/>Node.js :9877
    participant C as Docker容器集群
    participant S as SQLite数据库

    U->>B: ① 提交测试请求<br/>(目标URL + 需求)
    B->>D: ② 调用LLM解析需求<br/>generate_response()
    D-->>B: 返回结构化测试计划
    B->>M: ③ POST /jsonrpc initialize()
    M-->>B: Session建立成功
    B->>M: ④ POST /jsonrpc tools/call nmap
    M->>C: docker exec pentest-nmap ...
    C-->>M: 扫描结果JSON
    M-->>B: Nmap执行结果
    B->>M: ⑤ POST /jsonrpc tools/call sqlmap
    M->>C: docker exec pentest-sqlmap ...
    C-->>M: 注入检测结果
    M-->>B: SQLmap执行结果
    B->>D: ⑥ 调用LLM分析结果<br/>generate_report()
    D-->>B: Markdown格式报告
    B->>S: ⑦ save_report() 持久化
    B-->>U: ⑧ 返回完整报告
```
